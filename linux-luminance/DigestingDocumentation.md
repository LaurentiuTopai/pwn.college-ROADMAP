## `[folder-path] --printfile [file]`
**Categorie:** afisare

Afișează continutul fisierului curent introdule file

_Util cand vrei sa afisezi tot ce este intr-un file _

## `man [command]`
**Categorie:** informatii

_Util cand vrei sa vezi ce contine o anumita librarie sau ce face o anumita comanda_
_Baza de date pentru manual este salvata in /usr/share/man_

## `man -k [fisier] SAU apropos [fisier]`

**Categorie:** informatii 

_Afiseaza din baza de date pentru acel fisier informatiile utile_
